Labyrinth user guide

After starting the game, the language selector menu will welcome us. By moving arrows you can choose what language you want to play and then with Enter to confirm it

Once we have chosen the language, we will be greeted by the main menu. here we have the opportunity to generate an existing map, create a new one, choose a different language or exit the game. Also in the menu you can control by moving the arrows

If you want to select a map, you will appear in front of all files that may contain a map. By moving arrows and pressing ENTER we want to select the map we like and we have the opportunity to go back to the main menu.

If a coincidence of file is empty then you will then post an error message and we have the option to select a new map.





If we have succeeded in selecting any map for us, the map will appear in front of us and the instructions associated with it.

On the map we can move by moving the arrows. If you want to change a sympathetic track element for us, it is important for the cursor to be above the character we want to change. Then, by pressing the keys shown in the picture, we can place or delete different track elements



We have the opportunity to change the size of the track by pressing Enter. We will then have to give the new size of the track. It is important that the dimensions of the track should be width, height minimum width 10. if you enter a smaller number, you receive the following error message.



It is also important to mention that the program warns you when changing the size of the track, because if you want to reduce the size of the track, certain track elements will be lost and cannot be restored. The program warns us about this:

Also, if you do not want to change the dimensions of the track, the width and height must be set to 0. we can't leave any of them blank because the program won't let us go any further.



When you have finished editing the track, you can save it by pressing the f12 button.

.



If you do not want to save, but want to edit the track, you must select the back to Editor option. If you do not want to save the status of the track, you must select No and the program will take you back to the main menu.

If we have saved successfully, the program will let us know, and by pressing any button we will find ourselves in the main menu again.



If we want to create a new course, we must first specify the height of the course, and then its width. It is important to note here that the dimensions of the track must be at least 10, because otherwise the program will not allow us to create the map. Example:

In this case, you have to re-enter the dimensions of the track until you enter numbers greater than 10.



If we have created a map of the desired size, you will appear in front of the blank map.

In addition, it is important to note that if there is no room on the track or there is no entry and exit, the program will not allow you to save the map. This is worth paying attention to!
